# Style Guidelines

本文档定义案例和 gold response 的写作标准。

## 核心原则

场景应该因为情绪和关系逻辑是真实的而显得有生命力，而不是因为用了熟悉的戏剧化短语。

## 目标文体

- 自然的当代中文对话（pilot 默认）
- 情绪精确而非华丽
- 该克制时克制
- 该黑暗时黑暗，但不是为了黑暗而黑暗

## 应该做到

- 每句话都承载实际的场景信息——情绪、动机、压力、关系状态、潜台词
- 对话像这个角色在这个场景里真的会说的话
- 克制本身是 point 的时候保持克制
- 风格服务于意义，不替代意义

## 应该避免

### 网文腔污染 `webnovel_register_contamination`
"眸色暗了暗""那是猎人看见猎物的眼神"——这类库存短语语义模糊、情绪泛化、对网文残留过度拟合。出现在没有上下文支撑的地方时，应被视为质量失败而非风格选择。

### 美化的空洞 `aestheticized_vagueness`
听起来很沉很文学但实际上没说什么的句子。坏案例不应该能藏在质感后面。

### 借来的亲密 `borrowed_intimacy`
没有角色逻辑支撑的罐头式诱惑 / 执念 / 危险感措辞。

### 类型片覆盖 `genre_pastiche_overreach`
场景不应该因为模型在训练数据里见过这类 pattern 就自动进入"暗黑言情模式"或"猎物-猎手模式"。

## 关于黑暗内容

案例可以包含黑暗、占有性、强制性或心理不稳定的材料。但黑暗是内容，不是风格许可证。黑暗场景仍然需要连贯的动机、关系逻辑、可信的声音和语境意义。

## 底线

拒绝假强度。如果一句话只是因为闻起来像回收的网文才显得有戏剧感，那它可能不够好。目标不是泛泛的好看，而是 scene truth。
