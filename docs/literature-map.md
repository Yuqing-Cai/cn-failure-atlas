# Literature Map

本项目与以下公开工作相邻但不重叠。

## 邻近工作

**Spotting Out-of-Character Behavior** (Findings of ACL 2025) — 提出对 OOC 行为做原子级评估而非整体打分。对本项目的启发在于：虚构对话失败往往不是整段一致的，同一段回复可能部分保持角色、部分偏离。细粒度 OOC 标记的思路值得借鉴。

**DMT-RoleBench** (AAAI 2025) — 中文角色扮演的广谱评测，覆盖多种角色类型和多轮动态交互。确认了中文 RP 评测作为研究方向的合法性。本项目更窄：不评广义角色扮演能力，而是聚焦在情绪推理、潜台词、关系逻辑和微妙 OOC 失真上。

**Character-LLM** (EMNLP 2023) — 把角色模拟当成 agent 训练问题来处理，强调 profile、经历、情绪状态和知识边界。本项目不做训练，但在 case 设计中复用了"角色质量取决于比表层 persona 更多的东西"这一认识。

**Detecting Character Knowledge Errors** (EMNLP 2025) — 隔离了角色扮演中的知识边界问题。本项目把知识 / 世界观遵守作为评测维度之一，但不把它当全部——模型可以知识正确但场景失真。

## 本项目的定位

已有工作覆盖了：广义角色扮演能力、人设一致性、角色知识错误。本项目聚焦在一个更窄的缺口：

> 模型能否正确读懂虚构对话中的情绪、潜台词和关系逻辑，并在续写时不引入微妙的结构性失真——包括关系写平、心理咨询腔入侵、张力过早收束、叙事模板化等。

这些失败在用户端被广泛感知，但在公开评测语言中仍然缺乏足够的描述。
