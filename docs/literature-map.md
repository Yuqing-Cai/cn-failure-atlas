# Literature Map

这个项目和以下几个方向的公开工作相邻，但做的不是同一件事。

**细粒度 OOC 检测**（Findings of ACL 2025）提出对 OOC 行为做原子级评估而不是整体打分。这个思路对本项目有启发：虚构对话的失败往往不是整段一致的，同一段回复可能部分保持角色、部分偏离。

**中文角色扮演广谱评测** DMT-RoleBench（AAAI 2025）覆盖了多种角色类型和多轮动态交互，确认了中文 RP 评测作为研究方向的成立。本项目更窄——不评广义角色扮演能力，聚焦在情绪推理、潜台词、关系逻辑和微妙 OOC 失真上。

**角色模拟 agent 训练** Character-LLM（EMNLP 2023）把角色模拟当成 agent 问题来处理，强调 profile、经历、情绪状态和知识边界都很重要。本项目不做训练，但分类设计中复用了这个认识。

**角色知识错误检测**（EMNLP 2025）隔离了角色扮演中的知识边界问题。本项目把知识和世界观遵守作为维度之一，但不把它当全部——模型可以知识全对但场景仍然是假的。

**本项目的定位：** 已有工作覆盖了广义角色扮演能力、人设一致性、角色知识错误。本项目聚焦的是一个更窄但在实际使用中被强烈感知到的缺口——模型能不能在虚构对话中正确处理情绪、潜台词和关系逻辑，而不在续写中引入微妙的结构性失真。
